FreeDCam is an OpenSource Camera App

Issue Tracker: https://github.com/troop/FreeDCam/issues

Supported APIs: camera1 (has the most stuff working) camera2 (alpha state, don't expect too much) PlayMemories (Sony Camera Remote API)

supports Real3D for LG Optimus3D and 3DMax (With Orientation Hack for ported ICS)

Supported Manual Modes (Device depending and working for video recording too): -Exposure -Contrast -Brightness -Saturation -Sharpness -Zoom -Shutter speed -Focus

Modes (also Devices depending): -Flash -Color -ISO -Exposure -White Balance -Scenes -Focus -AntiFlicker -Scene Detection -PostProcess Denoise -Digital Image Stabilization -Memory Color Enhancement -Zero Shutter Lag (Must be disabled for RAW Capture)

Known Devices with RAW and DNG support LG G3/G2 (G2 has problems with leaked Lollipop!) Nubia NX503A Gionee Elife E7 Lenovo K910 Sony Xperia L HTC M8/M9 Yureka OnePlus One Htc One SV/XL Xiaomi_Redmi_Note Xiaomi_mi3 Alcatel Idol 3 Xiaomi Note Pro Vivo Xplay3s

Meizu MX4/MX5, THL5000, I Mobile I Style Q6 can convert raws taken with engineer tool to dngs with freedviewer

Theoretically, FreeDCam is working on any Qualcomm device that has raw/mipi/qcom in picture format list

FAQ:

Q:How can i capture RAW/DNG? A:In PictureFormat select one of the bayerformats/raw and set the Dng switch to DNG

Q:Why is my Device listed as supported but it dont create a Dng file? A:In that case we dont have added the BuildModel. To get it click on Save CamParameters and open the .txt file and send us the First line.

Q:My devices is not listet but im able to capture raw, can you add DNG support for it? A:Yes we can! Upload somewhere a raw and the Parameters and send us the link

If your Build Model is missing send me an email with that. Then dngconverting will work soon
